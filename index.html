HAND GESTURE RECOGNITION AND CURSOR CONTROL

This project aspires to create a hand gesture recognition system that can operate a computer cursor without using typical input methods. OpenCV, MediaPipe, and deep learning models (CNNs, RNNs) allow for real-time tracking and accurate recognition. Applications include assistive technology, gaming, virtual reality, and smart home control. The initiative improves accessibility, touchless interaction, and user experience, paving the way for more intuitive human-computer interaction in a variety of settings.
